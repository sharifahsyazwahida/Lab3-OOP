# Lab 3 - Matrix and Vector OOP System

## Description
This project implements a matrix and vector system using C# with Object-Oriented Programming principles including encapsulation, composition, aggregation, and dependency.

## Features
- Matrix operations (addition, multiplication, etc.)
- Vector calculations
- OOP design patterns demonstration

## How to Run
1. Open the solution file (.sln) in Visual Studio
2. Build the solution (Ctrl+Shift+B)
3. Run the project (F5)

## Class Structure
- **Matrix.cs**: Contains matrix operations
- **Vector.cs**: Contains vector operations  
- **Program.cs**: Main program entry point
- **MatrixRepo.cs** : contain matrix repository
- **VectorRepo.cs** : contain vector repository
- **Validator.cs** : contain validation of matrix and vector
- ** Calculator.cs** : contain the calculation of addition of matrix and vector

## Technologies Used
- C#
- .NET Framework
- Visual Studio
